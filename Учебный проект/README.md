# Система управления доставкой "Быстрая доставка"

Внутреннее приложение для учёта заказов с поддержкой CLI и GUI интерфейсов, экспорта/импорта данных и аналитики.

## Технологический стек
- Python 3.8+
- База данных: SQLite (с поддержкой внешних ключей `ON DELETE RESTRICT`)
- GUI: Tkinter
- Тестирование: pytest
- Логирование: встроенный модуль `logging`

## Установка
1. Клонируйте или скачайте проект.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Запуск:
Запустите графический интерфейс:
```bash
python main_gui.py
